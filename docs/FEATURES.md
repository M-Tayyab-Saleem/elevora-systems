# Features

Elevora Systems provides a comprehensive suite of tools designed to replace disjointed HR and project management applications.

## 1. Role-Based Access Control (RBAC)
The system is built on a strict permission hierarchy.
- **Super Admins:** Have global access. Can create managers, manage system-wide settings, and view all company data.
- **HR Managers:** Can onboard employees, approve/reject leave requests, and manage company announcements.
- **Employees:** Can log their time, request time off, submit expenses, and manage assigned project tasks.

## 2. People & Directory Management
- **Centralized Profiles:** A searchable directory containing contact information, department assignments, and reporting lines.
- **Document Management:** Securely upload and store employee documents (contracts, identification) linked directly to their profile.

## 3. Time & Attendance
- **Clock In / Out:** Real-time attendance tracking with daily timers.
- **Timesheets:** Employees can submit weekly timesheets detailing hours worked on specific projects.
- **Leave Management:** Employees can request sick leave, PTO, or unpaid leave. Managers receive notifications and can approve or reject requests via their dashboard.

## 4. Project Kanban
- **Visual Task Management:** Organize tasks using a drag-and-drop Kanban board interface.
- **Ticket Assignment:** Assign tickets to specific employees, set priorities, and track status (To Do, In Progress, Review, Done).

## 5. Expense Processing
- **Expense Submissions:** Employees can upload receipts and submit expense reports.
- **Manager Review:** Managers can review uploaded receipts and approve or reject expenses for reimbursement.

## 6. Live Notifications
- **System Alerts:** Real-time updates notify users when a leave request is approved, a ticket is assigned, or a global announcement is made.
