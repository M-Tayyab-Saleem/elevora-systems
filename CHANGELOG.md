# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Open Source Release

### Added
- Complete HRMS suite including People Management, Time Tracking, and Project Kanban.
- Interactive Demo Mode capabilities allowing recruiters and reviewers to explore the application without friction.
- Comprehensive demo data seeding scripts to populate the database with realistic dummy data.
- Professional portfolio documentation including Architecture and Deployment guides.
- Modern Landing Page redesign utilizing `framer-motion` and `lucide-react`.

### Security
- Scanned and removed all traces of internal client branding, sensitive configuration IDs, and debug logs to ensure a secure, clean open-source release.
